## 💡 추가하면 좋을 기능 제안

### 1. ⭐ 우선순위 (Priority) - 강력 추천

현재 작업들이 순서만 있고 우선순위가 없습니다.

**구현 아이디어:**
- High / Medium / Low 3단계 우선순위
- 컬러 또는 아이콘으로 시각적 구분
- 우선순위별 필터링/정렬
- Focus Mode에서 자동으로 높은 우선순위 작업부터 표시

**예상 난이도:** 중간  
**예상 개발 시간:** 2-3시간

---

### 2. 🏷️ 태그/카테고리 시스템

작업을 분류하고 영역별 시간 분석이 가능해집니다.

**구현 아이디어:**
- "Work", "Personal", "Study" 등 커스텀 태그
- 태그별 필터링
- Stats에서 태그별 시간 분석 차트
- 태그 컬러 커스터마이징

**예상 난이도:** 중상  
**예상 개발 시간:** 4-5시간

---

### 3. 📋 서브태스크 (Subtasks)

큰 작업을 작은 단계로 나눌 수 있습니다.

**구현 아이디어:**
- 메인 작업 하위에 체크리스트 형태로 추가
- 진행률 표시 (2/5 완료됨)
- 서브태스크 토글 펼침/접기
- 모든 서브태스크 완료 시 자동 완료 옵션

**예상 난이도:** 중상  
**예상 개발 시간:** 5-6시간

---

### 4. ⏱️ 예상 시간 설정

시간 추정 능력을 향상시킬 수 있습니다.

**구현 아이디어:**
- 작업 생성 시 예상 소요 시간 입력 (선택사항)
- 실제 시간과 예상 시간 비교
- 히스토리에서 정확도 통계
- "예상보다 2배 걸림" 같은 피드백

**예상 난이도:** 낮음  
**예상 개발 시간:** 1-2시간

---

### 5. ☕ 휴식 알림/Break Timer

포모도로 기법의 휴식 시간을 자동화합니다.

**구현 아이디어:**
- 25분 집중 후 5분 휴식 자동 제안
- 휴식 타이머
- 4세션 후 긴 휴식 (15-30분)
- 알림 소리/진동 옵션

**예상 난이도:** 중간  
**예상 개발 시간:** 2-3시간

---

### 6. 🎯 주간/월간 목표

장기적인 목표 관리가 가능해집니다.

**구현 아이디어:**
- 주간 목표 설정 (예: 이번 주 20개 완료)
- 주간/월간 리뷰 화면
- 목표 달성률 트래킹
- 주간 리포트 생성

**예상 난이도:** 중간  
**예상 개발 시간:** 3-4시간

---

### 7. 📝 노트/메모 기능 개선

현재 `notes` 필드의 활용도를 높입니다.

**구현 아이디어:**
- 마크다운 지원
- 작업 완료 후 회고 작성
- 노트 검색 기능
- 노트 미리보기

**예상 난이도:** 중간  
**예상 개발 시간:** 2-3시간

---

### 8. 💾 백업/내보내기

데이터 보관 및 이동이 가능해집니다.

**구현 아이디어:**
- JSON/CSV로 데이터 내보내기
- 데이터 가져오기
- 자동 백업 (로컬)
- 구글 드라이브 연동 (선택사항)

**예상 난이도:** 낮음-중간  
**예상 개발 시간:** 2-3시간

## 🏆 추천 우선순위

| 순위 | 기능 | 이유 |
|------|------|------|
| 1위 | **우선순위** | 가장 기본적이고 자주 쓰이는 기능 |
| 2위 | **태그/카테고리** | 생산성 앱의 핵심 기능 |
| 3위 | **예상 시간 설정** | 구현이 쉽고 실용적 |
| 4위 | **휴식 알림** | 포모도로와 시너지 |
| 5위 | **서브태스크** | 큰 프로젝트 관리에 필수 |
